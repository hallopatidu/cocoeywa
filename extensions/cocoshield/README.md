
# [Introduce](https://hsgdao1srjzq.sg.larksuite.com/wiki/WBwmwWQqFiCmIqkgIrcl50O8gQc)
# Installation
```
npm i
```

# Global Requirement
tsc - node module

# Build Case

### Build the extension running on the production environment ,include: enable obfuscation code, reopen cocos creator
```
npm run prod 
```
### Build the extension running on the development environment , include: enable obfuscation code, reopen cocos creator
```
npm run dev   
```
### Build the extension running on the development environment , include: enable obfuscation code, running in background
```
npm run dev:hide    
```
### Build the extension running on the development environment , include: Disable obfuscation code, reopen cocos creator
```
npm run debug  
```
### Build the extension running on the development environment , include: Disable obfuscation code, running in background
```
npm run debug:hide  
```
### Build the extension running on the production environment, just use for Jenkin build tool.
```
npm run build 
```
